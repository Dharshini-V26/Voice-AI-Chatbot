Setup Instructions 
1) Requirements

Python 3.8 or above, Microphone & Internet connection

2) Installation

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

3) Run

python chatbot.py

4) Usage Commands

   1) Open Google
   2) Open YouTube
   3) What is the time
   4) What is the date
   5) Wikipedia <topic>
   6) Write note
   7) Show notes
   8) Set reminder
   9) Show reminders
   10) Help
   11) Exit

5) Stop Program

Say: exit Or press: Ctrl + C
